# TWRP_Korean_Patcher는 무엇인가요?
TWRP_Korean_Patcher는 TWRP용 한국어 자동 패치 스크립트로, TWRP에 한국어 지원을 추가해요. 이 프로젝트는 Android Image Kitchen을 이용하여 자동으로 TWRP를 압축 해제하고, 필요한 패치 파일을 설치해요.

TWRP_Korean_Patcher는 현재 삼성 갤럭시용 .tar 형식 TWRP만 지원해요.

프로젝트에 대한 모든 형태의 기여, 제안, 버그 보고 또는 기능 요청을 환영해요.

# 기능
- TWRP의 폰트를 변경해요!
- TWRP에 한국어 지원을 추가해요!
- 자동으로 이미지를 압축 해제하고 빌드해요!
