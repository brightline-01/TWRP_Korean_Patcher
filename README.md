# TWRP_Korean_Patcher는 무엇인가요?
TWRP_Korean_Patcher는 TWRP 전용 한국어 자동 패치 스크립트로, TWRP에 한국어 지원을 추가합니다.
이 프로젝트는 BatchFile과 Android Image Kitchen을 이용하여 자동으로 TWRP를 압축 해제하고, 필요한 패치 파일을 병합합니다.

TWRP_Korean_Patcher는 현재 삼성 갤럭시용 .tar 형식, 범용 .img 형식 TWRP를 지원합니다.

프로젝트에 대한 모든 형태의 기여, 제안, 버그 보고 또는 기능 요청을 환영해요.

# 기능
- TWRP 기본 폰트 변경
- TWRP 한국어 지원 추가
- 이미지 자동 압축 해제, 자동 압축
- 원클릭 패치

# 어떻게 사용하나요?
1. Input_TWRP 폴더에 TWRP 파일(.tar 또는 .img)을 넣습니다.
2. Patch_TWRP.bat를 실행합니다.
3. Output_TWRP 폴더에 패치된 TWRP 파일이 출력됩니다.
